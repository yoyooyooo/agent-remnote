# Documentation / docs 导航

## Quick start / 最短阅读路径

1. `docs/ssot/`：single source of truth（裁决来源，与源码双向对齐）
2. `docs/ssot/agent-remnote/`：protocols & contracts（queue / WS / UI context / tool semantics）
3. `docs/guides/`：operations guides（debugging / validation / troubleshooting）
4. `docs/remnote/`：RemNote concepts + plugin SDK + local DB read-only notes
5. `docs/proposals/`：proposals and exploration（未定型方案）
6. `docs/architecture/`：future architecture notes（例如拆包路线图）
