# Security Policy

## Supported versions

This project is under active forward-only evolution. Security fixes are applied to the latest mainline only.

## Reporting a vulnerability

Please do **not** open a public issue for security vulnerabilities.

Report privately via:

- GitHub Security Advisories (preferred), or
- a private channel agreed with maintainers.

Include:

- affected component (`packages/agent-remnote`, `packages/plugin`, or docs/protocol)
- reproduction steps
- potential impact
- suggested remediation (if available)

We will acknowledge receipt, triage, and provide a remediation timeline.
