按技能里的顺序，最稳的检查路径是：

1. 先看 help，确认 `scenario run` 和 `scenario schema explain` 的当前参数面。
```bash
agent-remnote scenario run --help
agent-remnote scenario schema explain --help
```

2. 再查 builtin 清单，确认 builtin id、var 名和默认值。
```bash
agent-remnote --json scenario builtin list
```
这条 builtin 的静态定义是：
- `source_scope`，默认 `daily:past-7d`
- `target_ref`，默认 `daily:today`

3. 组装 dry-run，先看 lowering，再决定后续是否真的写入。
过去 3 天对应 `daily:past-3d`，范围不含今天。若你要把今天也算进去，用 `daily:last-3d`。

最终 dry-run 命令：
```bash
agent-remnote --json scenario run builtin:dn_recent_todos_to_today_move --dry-run --var source_scope=daily:past-3d --var target_ref=daily:today
```

补充约束：
- 不要用已退役的 `daily:previous-*`
- dry-run 不加 `--wait`
