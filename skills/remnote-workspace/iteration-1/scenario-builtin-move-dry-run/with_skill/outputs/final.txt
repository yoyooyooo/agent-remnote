这条 builtin 的 var 名是 `source_scope` 和 `target_ref`。默认值分别是 `daily:past-7d`、`daily:today`。按当前日期 2026-03-23 计算，默认 `daily:past-7d` 覆盖 2026-03-16 到 2026-03-22，不含今天。

按这版 `scenario` 实验面的 help-first 路由，最稳的检查顺序是：

1. `agent-remnote scenario run --help`
2. `agent-remnote scenario schema explain --help`
3. `agent-remnote --json scenario builtin list`
4. 如需再做一次 schema 级确认，先把 builtin 注入用户仓：`agent-remnote --json scenario builtin install dn_recent_todos_to_today_move --if-missing`
5. 再解释安装后的 package：`agent-remnote --json scenario schema explain --spec @$HOME/.agent-remnote/scenarios/dn_recent_todos_to_today_move.json`

你这次要“过去 3 天搬到今天”。按 2026-03-23 计算，`daily:past-3d` 覆盖 2026-03-20 到 2026-03-22，今天 2026-03-23 不在范围内。最终 dry-run 命令用这条，不加 `--wait`：

```bash
agent-remnote --json scenario run builtin:dn_recent_todos_to_today_move \
  --dry-run \
  --var source_scope=daily:past-3d \
  --var target_ref=daily:today
```

如果你想把今天也算进去，把 `source_scope` 改成 `daily:last-3d`。
